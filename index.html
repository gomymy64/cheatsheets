01

Установка Claude и Skill Creator в AntiGravity

Подключаем Claude к среде разработки и ставим плагин для создания скиллов

Открой AntiGravity → Терминал → Новый терминал
Скачай расширение Claude, введи команду из документации
Claude предложит выбрать: купить API-токены или использовать подписку. Выбери свой вариант
Нажми / → введи plugins → найди Skill Creator → установи
Нажми / → введи Enable → перезапусти среду если нужно
Нажми / → введи Skill Creator — готов к работе
02

Скилл #1 — SEO-оптимизатор статей

Создаём скилл, который берёт любую статью и выдаёт готовый к публикации текст, оптимизированный для топа Google

Вызови /Skill Creator и отправь промпт:
Промпт 1 — SEO-оптимизатор
Привет, создай мне скилл для SEO-оптимизации статей. Он должен срабатывать, когда я скидываю тебе статью. Задача — оптимизировать её так, чтобы она взлетела в топе Google. Ключевые слова подбирай сам. На выходе мне нужна готовая, вылизанная статья, которую можно сразу публиковать. Назови скилл 'SEO-оптимизация' и сохрани в глобальный каталог навыков
Протестируй на любой статье, подправь если нужно
Скилл будет доступен во всех приложениях Claude (десктоп, терминал и т.д.)
03

Скилл #2 — Генератор инфографики (Nano Banana 2)

Создаём скилл для генерации картинок в едином фирменном стиле через API Kie.ai

Подготовь системный промпт со своим брендбуком (шрифты, цвета, стилистика). Лайфхак: найди крутую инфографику в сети и закинь как референс
Зайди на kie.ai → скопируй страницу kie.ai/nano-banana-2
Вызови /Skill Creator и отправь:
Промпт 2 — генератор инфографики
Привет. Создай для меня скилл и сохрани его в глобальный каталог. Ты будешь использовать внешнее API (данные я сейчас дам). Твоя задача: генерировать для меня инфографику в формате 16:9 строго по моим гайдлайнам. Работать это будет так: я кидаю тебе сырой текст или набор тезисов, а ты превращаешь их в красивую инфографику. Картинки должны быть визуально привлекательными и читаемыми — больше схем, меньше голого текста
Вставь в чат скопированную страницу Nano Banana 2 и свой API-ключ от Kie.ai
Убедись что выбрана модель Nano Banana 2, версия 2K, пропорции 16:9
Nano Banana 2  |  2K разрешение  |  16:9
Для загрузки картинок Claude нужен публичный URL. Используй kie.ai — скорми Claude API-ключ оттуда, и он сам будет загружать картинки и получать прямые ссылки.
04

Скилл #3 — Лид-магнит из YouTube в Notion

Берём URL видео → вытягиваем транскрипт → превращаем в пошаговый гайд в Notion

Вызови /Skill Creator и отправь:
Промпт 3 — лид-магнит YouTube → Notion
Я хочу создать навык. Он будет получать от меня URL-адрес любого видео на YouTube. Задача скилла: перейти по ссылке, спарсить видео через субтитры, вытащить транскрипт, а затем превратить его в крутое пошаговое руководство. Результат — новая страница в Notion, красиво оформленная со всеми фишками Notion. Стиль — максимально практичный, пошаговые инструкции, без воды и клише. Не используй дефисы в списках. Добавляй обложку к каждому гайду. Используй мой tone of voice. Сохрани навык в глобальный каталог
05

Скилл #4 — RAG-бот с векторной памятью (Pinecone)

Парсим YouTube-канал → векторизуем в Pinecone → общаемся с ботом, который знает весь контент

🔑 Подготовка — YouTube Data API

Зайди на console.cloud.google.com
Открой раздел проектов → в поиске найди «YouTube Data API v3»
Нажми «Включить» (Enable)
Слева → «Учетные данные» → «Создать учетные данные» → «API-ключ»
Скопируй ключ

🌲 Подготовка — Pinecone

Зайди на pinecone.io → создай бесплатный аккаунт
Сгенерируй API-ключ

⚡ Подготовка — OpenAI API

Зайди на platform.openai.com → создай API-ключ
Вызови /Skill Creator и отправь:
Промпт 4 — RAG-бот
Привет. Я буду давать тебе URL-адрес YouTube-канала. Ты должен взять 10 последних видео этого человека, векторизовать их транскрипты и сохранить в векторную базу данных Pinecone. После этого ты создашь чат-бота, с которым я смогу вести диалог и задавать вопросы на основе контента этого канала. Ключ API от Pinecone я тебе дам. У тебя уже есть скилл для парсинга YouTube — можешь объединить их в один процесс
Скорми Claude API-ключи: YouTube, Pinecone, OpenAI
модель для векторизации — multilingual-e5-large
06

Скилл #5 — Генератор счетов (инвойсов)

Автоматическая генерация красивых HTML-счетов на оплату

Вызови /Skill Creator и отправь:
Промпт 5 — генератор инвойсов
Привет. Создай мне скилл, который будет генерировать счета на оплату (инвойсы). В каждом счете жестко пропиши мой адрес: Пушкина 9, город Москва. Результат должен выдаваться в виде красивого одностраничного HTML-документа, который строго помещается на одной странице при печати — никаких переносов на второй лист. На вход я буду давать тебе только название компании-получателя, их адрес и сумму, а ты сам соберешь из этого готовый счет. Вставь в HTML мою фотографию из папки 'Рабочий стол/изображения' под названием 'ЛОГО', обрезанную в форме круга. Без лишних отступов по краям. Сделай всё красиво
Скидывай Claude скриншоты результата и вноси правки визуально.
07

Скилл #6 — Конкурентная разведка (Firecrawl)

Вводишь название бренда → получаешь полный аналитический отчёт

Зайди на firecrawl.dev → получи API-ключ
Вызови /Skill Creator и отправь:
Промпт 6 — конкурентная разведка
Привет. Создай скилл для конкурентной разведки. Я буду давать тебе название бренда или URL-адрес. Твоя задача — используя Firecrawl, глубоко и дотошно просканировать этот проект. Мне нужно всё, что ты найдешь. Узнай, кто их ключевые сотрудники, найди о них ту информацию в сети, о которой они сами давно забыли. Раскопай историю создания компании и любые ценные инсайты. И в конце упакуй всё это в один ясный, четкий и визуально красивый отчет

💡 Лайфхаки из видео

Скиллы бывают двух типов: «расширение возможностей» (устареют с новыми моделями) и «закодированные предпочтения» (твой уникальный процесс — останутся навсегда)
Всегда тестируй скилл перед сохранением — Skill Creator умеет прогонять внутренние тесты
Можно скидывать скриншоты для визуальных правок
Модель multilingual-e5-large — лучший выбор для векторизации в Pinecone
Скиллы портативные — работают во всей экосистеме Anthropic
Из одного куска контента можно делать лид-магниты для любой платформы
